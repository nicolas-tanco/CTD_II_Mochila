1) Git fue crado por Linus Torvalds al tener problemas con los sistemas de control de version de ese momento. Su objetivo fue desarrollar un sistema que supere a los existentes y que resuelva sus problemas de performance y escabilidad para que Linux pueda seguir siendo desarrollado.
2) Github fue comprada por Microsoft en el 2018 con el objetivo de aumentar su uso empresarial y llevar las herramientas y servicios de desarrollo de Microsoft a nuevas audiencias.
3) Se puede trabajar directamente desde github subiendo archivos manualmente.
